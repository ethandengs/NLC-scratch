
# 🐑 Sheep Care Game

一款療癒的牧羊養成遊戲。

## 遊戲特色
- **領養**: 召喚迷途的小羊。
- **禱告**: 透過禱告 (點擊) 照顧小羊，讓牠們健康長大。
- **成長**: 小羊會進化 (小羊 -> 忠信羊 -> 黃金公羊)。
- **動態**: 小羊會在牧場上自由漫步。
- **風險**: 如果不好好照顧，小羊會生病甚至逃跑！
- **管理**: 使用「牧羊人筆記」手動調整小羊狀態 (測試用)。

## 執行方式

1. **安裝依賴**:
   ```bash
   npm install
   ```
   (如果遇到權限問題，請嘗試使用 `npm.cmd install`)

2. **啟動遊戲**:
   ```bash
   npm run dev
   ```

3. **遊玩**:
   打開瀏覽器訪問顯示的網址 (通常是 `http://localhost:5173`)。

## 操作說明
- **點擊小羊**: 禱告（恢復健康、增加成長值）。
- **點擊鈴鐺**: 召喚新小羊。
- **點擊鉛筆圖示**: 開啟編輯器 (Debug Tool)。
- **點擊 Shepherd**: 收藏黃金公羊。
